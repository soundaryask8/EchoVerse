# EchoVerse 🌙

EchoVerse is a simple blogging web application built using Node.js, Express.js, EJS, HTML, and CSS.

The application allows users to create and view blog posts through a clean and responsive interface.

## Features

* Create new blog posts
* View all published posts on the homepage
* Responsive design for mobile and desktop devices
* Reusable EJS partials for header and footer
* Modern UI with a gradient background and glassmorphism-inspired design

## Technologies Used

* Node.js
* Express.js
* EJS
* HTML5
* CSS3

## Current Limitations

This version does **not use a database**.

All blog posts are stored temporarily in memory while the server is running. Once the server is restarted, all posts are lost.

## Future Improvements
